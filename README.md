# Sri Balaji Mill and Industry

Welcome to the official repository for Sri Balaji Mill and Industry website.

## Live Demo

Check out the live website here: [https://sri-balaji-mill.vercel.app/](https://sri-balaji-mill.vercel.app/)

## Features

- **Multilingual Support**: Available in English, Telugu, and Hindi.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Product Showcase**: Detailed information about our various divisions and products.
- **Contact Integration**: Easy click-to-call features and location details.

## Tech Stack

- React
- Vite
- Vercel (Deployment)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
